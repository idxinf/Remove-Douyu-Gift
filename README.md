# Remove Douyu Gift

`Remove Douyu Gift`是一款基于`JavaScript`开发的`TamperMonkey`（油猴）脚本，用于隐藏斗鱼直播间的礼物栏以及优化工具栏的显示效果。

----------

### 安装
#### :monkey: 安装油猴
油猴插件可以通过以下方法进行安装：
1. 通过浏览器的扩展商店安装：[Chrome扩展](https://chromewebstore.google.com/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo) / [Edge扩展](https://microsoftedge.microsoft.com/addons/detail/iikmkjmpaadaobahmlepeloendndfphd) / [Firefox扩展](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey)
2. 手动下载并将安装包拖动到浏览器安装：[油猴crx安装包](https://www.tampermonkey.net/crx/tampermonkey_stable.crx)

#### :page_facing_up: 安装脚本
通过Greasy Fork安装：[Remove Douyu Gift](https://greasyfork.org/zh-CN/scripts/563628)
